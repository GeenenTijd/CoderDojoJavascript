for( 1 ; 2 ; 3 ){}

  • 1 is het start gedeelte, wat moet er gebeuren voordat de lus start. Hier maken we een variabele i.
  • 2 is de controle of de lus mag verder doen of niet. Zolang i kleiner is dan 3 doen we verder.
  • 3 is wat er moet gebeuren wanneer we aan het einde van de code in de lus komen. Hier tellen we 1 op bij i (++i is hetzelfde als i = i + 1).
